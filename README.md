# AI-Agent-for-Industrial-Anomaly-Detection-in-Smart-Manufacturing
AI-based agent for anomaly detection in smart manufacturing, focused on carpet defects. It analyzes sensor and process data to identify irregular patterns, weaving issues, and abnormalities. The system enables real-time monitoring, reduces manual inspection, and improves quality control and efficiency.
